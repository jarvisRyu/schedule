# 🗓️일정관리 앱 API

## 🛠️주요 기능
* 일정 등록 
* 전체 일정 조회
* 이름 또는 날짜 기준 조회
* 일정 단건 조회
* 일정 수정 
* 일정 삭제 

## 🛠️작업 환경
* JAVA 17+
* Spring Boot
* dependencies : Lombok, Spring Web,JdbcTemplate

## 📁 프로젝트 구조 
```
└─schedule
     ├─controller
     ├─dto
     │  ├─request
     │  └─response
     ├─entity
     ├─repository
     ├─scheduleexceptionhandler
     └─service
```
